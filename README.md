# Тестовое задание: Детекция стен

## Этапы пайплайна:
1. **Подготовка данных.** Разметка изображений (аннотации bounding boxes для стен) и создание датасета в формате YOLO.
2. **Обучение модели.** Обучение модели YOLOv8n на тренировочном датасете. Получена модель `baseline_model.pt`.
3. **Работа прототипа (скрипт `prototype.py`):**
   - Загрузка обученной модели и библиотек.
   - Получение входных данных от пользователя (ввод с клавиатуры. Путь до картинок в формате `.jpg` или `.png`).
   - Получение результата работы модели.
   - Конвертация координат bounding boxes в JSON формат и сохранение в файл.
   - Наложение bounding boxes на исходное изображение и сохранение в директорию `output_images`.

## Какие модели/инструменты использованы и почему
Для разметки bounding boxes использовался инструмент CVAT. Он является opensource, прост в применении и имеет возможность экспортирования размеченных изображений в формат датасета YOLO.

Для модели детекции использовался YOLOv8n. Она проста в использовании, быстро дообучается и хорошо работает на небольших датасетах.

## Где слабые места
1. Модель демонстрирует метрики, которые не подходят для использования в продуктах на коммерческой основе (mAP50 ≈ 0.57, mAP50-95 ≈ 0.35).
2. Модель работает только с горизонтальным или вертикальным изображением стен.
3. Отсутствие пользовательского интерфейса.
